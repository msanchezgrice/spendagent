§02 / THE AUDIT

Four steps. One Markdown verdict. No dashboards. No uploads.

01✧
Walk the repo.
Regex + heuristics find every LLM callsite in TS, JS, and Python — OpenAI, Anthropic, Gemini, LiteLLM, Vercel AI, LangChain, LlamaIndex, embeddings. .gitignore is respected.
$ agentspend scan --repo .
02
Import your usage.
A plain CSV: provider, model, input_tokens, output_tokens. Optional route, feature, cost_usd. If you have it, we use it; if not, we show scenario estimates.
$ agentspend import-usage usage.csv
03
Match code to money.
We map each usage row back to a callsite by route, feature, model, and filename — with explicit high / medium / low mapping confidence, never guessed dollars.
$ agentspend analyze
04
Read the verdict.
A ranked list of findings with estimated savings, effort, risk, alternatives, and a small safe patch plan — written to .agentspend/AGENT_SPEND_REPORT.md.
$ agentspend report

OR RUN THE WHOLE PIPELINE AT ONCE: $ agentspend doctor --repo . --usage ./usage.csv

§04 / WASTE PATTERNS

Eight patterns. All of them are in your repo.

/ 01 Model Switch

Premium model on simple task

Sonnet, Opus, gpt-4o, o1 used for classification, extraction, or summarization routes where cheaper alternatives exist.

/ 02 Prompt Cache

No cache on deterministic calls

Temperature 0, repeated inputs, identical system prompts — paid for on every request.

/ 03 Fanout

LLM call inside a loop

for, .map, Promise.all, list comprehensions — multiplied by N inputs with no batching.

/ 04 Embeddings

Embedding recomputation

Embed-in-loop with no persistence, no content hashing, no batch API.

/ 05 Batching

Background jobs at realtime prices

Calls in jobs/, cron/, workers/, scripts/ — batch API is ~50% cheaper.

/ 06 Context Bloat

Large global agent context

CLAUDE.md, AGENTS.md, .cursor/rules/ — if it’s >4k tokens, every session pays the toll.

/ 07 Agent Tooling

MCP server sprawl

Many MCP servers configured. Every tool description rides in context by default.

/ 08 Telemetry

Missing feature tags

No metadata, user, or tags on calls. Future spend will be impossible to attribute.
